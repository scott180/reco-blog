    # 一、名称说明

    无为徐生
    无为徐生是本人昵称，笔名，也是今日头条号、微信公众号、百度百家号的名称。
    之所以叫这个名字，是根据本人的籍贯、思想、性别、性格等等综合条件来的。

    无为是本人老家，安徽无为，风景秀丽，历史悠久。
    无为也是一种道家思想，顺其自然，因时而行。
    摒弃妄自作为，远祸慎行，追求朴素节俭、清静寡欲的境界。
    徐是本人的姓，男生，所谓斯文有礼谓之生，故名徐生。

    综上种种，得名-无为徐生，希冀无为有为，徐生长生。
    人生在世，有所作为，文化长生，思想永恒。
    --202108

    # 三、徐书法

    学以致用，见贤而思齐。
    之前《书法练习轨迹》都是保存在第三方平台，自己不能随心所欲的记录、更新。
    本人是一个程序员，竟然没有属于自己的网站，资料不在自己的服务器，有点说不过去啊。
    因此在腾讯云购买了一个轻量应用服务器且注册了新域名：xushufa.cn (opens new window) ，以后这就是我的书法记录平台。
    之所以叫：xushufa，中文：徐书法。是截取本人笔名【无为徐生】中的“徐”及书法练习记录【书法练习轨迹】的“书法”组合成【徐书法】。

    【徐书法】含义有三：
    一、表示是徐生的书法，徐某人的书法。
    二、人生如练字，需要不疾不徐，不骄不躁。提醒自己要慢下来，静下去。
    三、古有“书圣”“颜体”，高山仰止，今有“徐书法”见贤思齐。十年如一日，徐徐练习，徐书必有法。

    --202203
