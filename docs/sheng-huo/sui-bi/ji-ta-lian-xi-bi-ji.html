    # 1 说明

    吉他（意大利语：Chitarra），又译为结他或六弦琴。是一种弹拨乐器，通常有六条弦，形状与提琴相似。

    吉他在流行音乐、摇滚音乐、蓝调、民歌、佛朗明哥中，常被视为主要乐器。
而在古典音乐的领域里，吉他常以独奏或二重奏的型式演出；当然，在室内乐和管弦乐中，吉他亦扮演着相当程度的陪衬角色。
古典吉他与小提琴、钢琴并列为世界著名三大乐器。

    # 2 课程

    # 2.1 第一节课

    第一节课 认识吉他、坐姿、弹弦、节奏。

    • 构成
      琴头，琴颈，琴身
      琴钮、弦枕、指板、琴弦、品线、品柱、琴桥、琴码、面板、音孔。

    • 坐姿
      右肘贴面板 凹处放右腿

    • 琴弦
      从细到粗一到六根弦
      无名指弹一弦
      中指弹二弦
      食指弹三弦
      大拇指弹四五六弦

    • 三个节奏 两天时间 每天三小时
      6321 5321 4321
      632123 532123 432123
      6323 5323 5323 5323 4323 5323

    • [ ] 学于 20230527 周六

    感想与笔记

    • 练 熟能生巧
      --20230527

    • 课程老师说那三个节奏要练两天时间，每天三小时。
      貌似做不到啊，一个是没时间，这是借口，时间就像海绵里面的水，只要挤挤总会有的。周末或者清晨夜晚都可以练一练。
      二是初次练习坚持不了那么长时间，才练二三十分钟就感觉腰酸腿疼眼花，尤其是近视眼眼干眼涩尤甚。这才是做不到的主因。
      个人觉得初次只练习第一个节奏 6321 5321 4321 ，而且练十分钟稍微停一下歇一下，升升懒腰，扭扭脖子，眺望眺望远方。
      初次练三四个十分钟就行了，早中晚各练一遍最好。直到眼睛不用看琴弦手指就可以下意识肌肉记忆识别六根弦就可以进行下一步了。
      关键在于坚持，保持兴趣，随心而为，尽力而行。
      --20230529

    # 2.2 第二节课

    第二节课 爬格子 训练手指拉伸度，力度，灵活度。

    • 变调夹，夹住吉他的第五品，靠近品线，不能压到品线。

    • 左手不能留指甲，大拇指正面顶住琴脊。
      大拇指可以上下左右移动。

    • 爬格子，指尖按住琴弦，垂直纸板，按在格子里面。
      靠近品线，不能按到品线，用力按下去。

    • 食指按第一品第一弦，变调夹的品，即原第六品。
      中指按第二品第二弦。
      无名指按第三品。
      小拇指按第四品。

    • 花费大量的时间，慢速度练习。
      初学者，每天一小时，练习二十天，就不会有沉闷的声音。爬格子这一关过了就成功了百分之八十。

    • [ ] 学于 20230604 周日

    感想与笔记

    • 爬格子有点找罪受挨酷刑的感觉，按一会儿，指尖就很疼。果然，高手大师不是那么轻易就能达到的。

    • 宝剑锋从磨砺出，梅花香自苦寒来。吉他不易，且行且努力

    • 没有捷径，只有方法。没有借口，只有练习。爬格子比较难。四个手指按住琴弦不容易。按住之后弹出响亮的琴音很难，切换琴弦流畅的变化旋律更难。需要在网上多找找爬格子的视频，学习正确的科学的有效的方法。但是关键在于每日练习。
      --20230605

    • 现在基本是忙里偷闲早晚十分钟爬格子。果然每日练习时长短了，虽然已过二十多天，但是目前爬格子还是比较生疏。

    • 只能把吉他当做兴趣爱好，有空了，想到了就练练。尽量做到每日练习，不荒废，不放弃。

    • 总结一下为什么不能长久坚持的原因：
      1、爬格子太疼，不想受酷刑。而且因为是程序员需要敲键盘，不能因为弹吉他手疼影响工作。（是借口也是正当理由。）
      2、没有同道师友，个人练习随心所欲信马由缰。想练就练，想不练就不练。无所谓，谁会对上谁。
      3、没有目标或者说是虽然树立了目标但是由于内因外因没有达成。
      4、练吉他只是爱好，不是吃饭的工具，没有紧张感紧迫感。
      5、人生苦短，良辰美景不可辜负。
      6、懒，没有毅力。
      --20230628
