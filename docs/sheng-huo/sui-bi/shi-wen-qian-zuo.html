    # 诗文浅作


    # 1、说明

    本文是我的诗词杂文短句小集，记录灵感一闪及咬文嚼字的诗文浅作。

    二零二零年九月至二零二二年十月，这两年空闲时间，下班及周末在自学书法，练习毛笔字。前期还算比较认真，后面渐渐半途而废。
    虽然书法没练好，但是记录练习书法的方式相当丰富。图片、文章、诗词、博客、社区、公众号、个人网站，每周拍练字照片写人生感想发到网站。文章写了不少，程序员的本职工作也得到了一定程度的锻炼。
    《书法练习轨迹--明月几时有》记录本人每周练习书法情况，包含练习图片及一些感悟。
    《无为徐生》，本人笔名，表示顺其自然，因时而行。
    《徐书法》是徐生的书法，徐某人的书法。
    《笔名汉字频率分析》使用java分析本人笔名昵称的分类数量及使用频率。
    具体内容可见gitlab的仓库 calligraphy (opens new window) ，2021到2022年更新较多。

    二零二三及二零二四年沉迷原神，不可自拔。作文《我的原神旅行观测记录》、《纪念一下，三百天打满深渊》等记录在document (opens new window)仓库。

    二零二五是新的一年，是平凡的一年，也是充满挑战的一年。


    # 2、诗词

    半生有感
    龙骨山上风车横，转去庐州蜀山风。
    十年一觉世不同，西湖水清临安梦。
    --202105

    注解：
1、家乡安徽无为有座龙骨山，山上有许多巨大的风力发电机，像大风车一样。
2、大学在合肥就读，合肥古称庐州，历史悠久，学校附近有个大蜀山，风景优美。
3、十年，虚指。2010年去合肥读大学，19年来杭州，作此诗是21年。物是人非，时移世易。
4、常去西湖游览，观赏山清水秀。定在临安居住，实现人生梦想。
--20250529

    元旦三天打油诗
    元旦三天，内外兼修。
    其第一日，三五好友。
    拜上天竺，观法喜寺。
    游人如织，车水马龙。
    其第二日，九州街上。
    临安文体，锦南宝龙。
    繁华似锦，自然和谐。
    其第三日，修养身息。
    阳光正好，晒被刷鞋。
    听歌悦耳，练字修心。

    水滴石穿，铁磨针成。
    人生在世，有为无为。
    打油之诗，摸鱼之工。

    运到功成，顺其自然。
    自知者明，知足常乐。
    --20220103

    2025有感
    世界大势乱如麻，媒体信息像喇叭。
    如何辩真做归纳，史海能钓未来虾。
    --20250523

阿美利卡也上台一位特不靠谱，世界大势纷乱如麻，媒体信息像喇叭一样繁杂。
真假难以分辨，敌我难以区分，后果难以预料。
如何去伪存真，只能参考历史，方可预测未来。
所谓，以史为鉴，可以知兴替，是也。
--20250529

    # 3、杂句

    我的三观：制恒强变真

    我的三观：人生观、世界观、价值观，亦即人生信条、五字真言：制 恒 强 变 真。
    制：制怒、自制、克制，不冲动、不贪婪，任何时候都要保持冷静，泰山崩于前而色不变。心平气和才能修身养性。
    恒：持之以恒，学习任何知识、技能都要长久的奋斗，坚持就是胜利。
    强：坚强、自强、自信、自立。
    变：变化，见机行事，随机应变。世上唯一的不变就是变化。
    真：真诚、真实。待人以诚、待己以真。认识自己，了解自己的优点缺点，知道自己要干什么。世上乱花迷人眼，需要仔细甄别，方能去伪存真。

    前三条（制、恒、强）是对内，对自己，后两条（变 真）是对外，对别人。只有内外兼修，审慎笃行，两手都要抓，两手都要硬，才能辨别真伪，提高自身。
    若感觉观点太多，信条很杂，只需要记住一个字：制。即可。

    在高中的某一天，突然醒悟了以上五字真言，这些年为人处世基本照此原则，不破藩篱。
    第一条-制怒，实践的比较好，一直都比较冷静平和。
    后三条-强、变、真，也算是身体力行。
    第二条-恒 。。。 世间之事，知易行难。行百里者，半途而废者九十。小丑竟是我自己。(⊙︿⊙)
